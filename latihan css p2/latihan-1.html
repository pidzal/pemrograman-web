Belajar CSS Dasar

Pendahuluan

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mempercantik tampilan halaman web. Dengan CSS, kita bisa mengatur warna, ukuran font, jarak, dan banyak lagi.

Kenapa Harus Belajar CSS?

  • Membuat tampilan web lebih menarik
  • Mengatur layout halaman dengan rapi
  • Memisahkan antara struktur (HTML) dan tampilan (CSS)

Kesimpulan

CSS adalah bagian penting dari pembuatan web. Dengan menguasainya, kita bisa membuat website yang tidak hanya berfungsi tapi juga enak dipandang.
